# Claim Status

This file summarizes the current epistemic status of major DT/FDS claims.

Status labels:

- **Core**: formal FDS architecture.
- **Physical bridge**: requires explicit physical implementation assumptions.
- **Domain bridge**: requires domain-specific mapping assumptions (life, cognition, economics, etc.).
- **Complex-systems bridge**: requires complex-systems mapping assumptions.
- **Operational**: testable in AI / complex systems settings.
- **High-risk**: speculative bridge claim, quarantined from the core.
- **Formal bridge / operational bridge**: spine paper bridging formal core to operational domains.
- **Frozen**: timestamped conceptual line, not actively developed in public repo.
