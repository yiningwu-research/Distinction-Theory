# Start Here

Distinction Theory (DT) is a broad claim-space archive.
Active Finite Distinction Systems (FDS) is the formal finite-system core.

If you are new, do not start with the full DT Archive.
Start with the FDS Core.

## 1. Five-Minute Entry

Read this path if you want the shortest route:

1. README.md — project scope and public layers.
2. FDS-0 abstract — formal finite-system core.
3. FDS-A1 abstract — artificial agency criterion.
4. CLAIM_STATUS.md — which claims are core, bridge, or high-risk.

Core idea:

finite systems that maintain boundaries under limited capacity must pay maintenance costs.
