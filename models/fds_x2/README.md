# FDS-X2: Three Fermion Generations as CP/T-Asymmetric Identity Transformation

## Contents

- `generate_results.py` — deterministic algebraic and normal-form demonstrations
- `fig1_ckm_phase_count.pdf` / `.png` — CKM phase count vs N
- `fig2_phase_bookkeeping.pdf` / `.png` — phase bookkeeping
- `fig3_hard_soft_map.pdf` / `.png` — hard lower bound vs soft minimality bridge
- `fig4_toy_flavor_cost.pdf` / `.png` — toy flavor-cost functional
- `fig5_chain_diagram.pdf` / `.png` — X2 dependency chain
- `data/*.csv`, `data/simulation_parameters.json` — generated output data

## Usage

```bash
pip install numpy pandas matplotlib scipy
python generate_results.py
```

All figures are deterministic algebraic or normal-form demonstrations, not empirical fits.
