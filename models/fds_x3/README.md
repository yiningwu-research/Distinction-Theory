# FDS-X3: Functional Decomposition of the Four Fundamental Interactions

## Contents

- `generate_results.py` — deterministic normal-form demonstrations
- `fig1_closure_square.pdf` / `.png` — four-operation closure square
- `fig2_coverage_matrix.pdf` / `.png` — qualitative coverage matrix
- `fig3_remove_one_failure.pdf` / `.png` — remove-one-operation test

## Usage

```bash
pip install numpy pandas matplotlib scipy
python generate_results.py
```

All figures are conceptual / normal-form demonstrations, not empirical fits.
